## Overview

The BMI Management System is a Python program that calculates a user's Body Mass Index (BMI) based on their height and weight. It also classifies the user's BMI into a health category.

This project was created as part of a Computer Fundamentals course to practice Python programming concepts such as functions, conditional statements, loops, input validation, and basic calculations.

## Features

* Enter a user's weight and height
* Calculate BMI using the standard formula
* Display the BMI value
* Classify BMI into a health category:

  * Underweight
  * Healthy Weight
  * Overweight
  * Obese
* Validate user input to ensure realistic values

## BMI Formula

BMI = Weight (kg) ÷ Height² (m²)
